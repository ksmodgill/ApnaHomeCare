# Apna Home Care Nursing Services

Premium home nursing services website for Apna Home Care Nursing Services, Muzaffarpur, Bihar.

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (animations)
- **Lucide React** (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Features

- Premium healthcare UI with conversion-focused design
- Mobile-first responsive layout with sticky CTA bar
- 8 service location pages for local SEO
- Blog structure for content marketing
- Complete SEO: LocalBusiness, MedicalBusiness, FAQ, Breadcrumb schema
- XML sitemap and robots.txt
- Open Graph and Twitter meta tags
- Floating WhatsApp and click-to-call buttons
- Inquiry forms for lead capture

## Deployment

Deploy to Vercel, Netlify, or any Node.js hosting platform. Update `SITE.url` in `src/lib/constants.ts` with your production domain.
